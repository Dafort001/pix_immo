# HTML-Seiten Download

## Schnellstart

### 1. Server starten
```bash
npm run dev
```

### 2. Seiten exportieren
```bash
./scripts/download-pages.sh
```

### 3. Herunterladen
```bash
# Die ZIP-Datei finden Sie hier:
export/piximmo-pages.zip
```
